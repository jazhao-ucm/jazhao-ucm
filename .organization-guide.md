# Organization Guide

This repository is organized as follows:

## Folder Structure

```
management-global-market-notes/
├── notes/                    # Class notes organized by date
│   └── YYYY-MM-DD-class-notes.md
│
├── study-guides/             # Chapter study guides
│   └── chapter-X-complete-study-guide.md
│
├── materials/                # Course materials
│   └── slides/              # PDF slides and presentations
│       └── *.pdf, *.pptx
│
└── README.md                # Main navigation and study guide
```

## Where to Put New Files

- **New class notes**: Add to `notes/` folder with format `YYYY-MM-DD-class-notes.md`
- **New PDF slides**: Add to `materials/slides/` folder
- **New study guides**: Add to `study-guides/` folder
- **Other materials**: Add to appropriate subfolder in `materials/`

## Quick Organize Command

If you add new files to the root directory, run:
```bash
# Move PDFs to slides folder
mv *.pdf materials/slides/ 2>/dev/null

# Move new class notes
mv 2026-*.md notes/ 2>/dev/null

# Move chapter guides
mv chapter-*.md study-guides/ 2>/dev/null
```
