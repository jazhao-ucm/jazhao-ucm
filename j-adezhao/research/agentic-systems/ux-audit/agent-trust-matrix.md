# Matrix Fidei Agentis (Agent Trust Matrix)

Cross-cultural trust evaluation scaffold for agentic interfaces.

## Dimensions

| Dimension | Prompt Design Signal | UI Signal | Metric |
| :--- | :--- | :--- | :--- |
| Transparency | Explain rationale on demand | “Why this?” toggle | Explanation usefulness score |
| Control | User can confirm/override | Explicit action gates | Override frequency |
| Cultural tone fit | Context-sensitive phrasing | Tone profile selector | Trust delta by cohort |
| Safety clarity | Risk warnings are explicit | High-risk badge | Unsafe action attempt rate |
| Reliability | Consistent outcomes | Status timeline | Task completion consistency |

## Cohort Lens

- Cohort A: local Spanish-context users.
- Cohort B: Asian or multilingual-context users.
- Cohort C: mixed professional users (cross-cultural teams).

## Rating Scale

- 1 = low trust
- 3 = moderate trust
- 5 = high trust

## Notes

Use this matrix together with `le-chat-dossier.md` and your validation protocol.
