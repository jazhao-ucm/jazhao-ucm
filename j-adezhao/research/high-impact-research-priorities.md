# Prioritates Investigationis (High-Impact Research Priorities)

Strategic research focus map aligned to your UX, AI, cross-cultural, and inclusion goals.

## I. Theoria-Machina (Agentic AI + UX: Human-Agent Loop)

**Why this is high-impact**

- Strong fit with your platform design background and iterative product workflow.
- High leverage for trust, adoption, and explainability in real deployments.

**Focus areas**

- Explainability in agentic workflows
  - Core question: how should agents expose reasoning to non-technical users without cognitive overload?
  - Method: adaptive transparency prototypes (for example, “show why” layers).
- Cultural context in agent behavior
  - Core question: how does perceived autonomy vary across user cultures?
  - Method: compare acceptance and trust in culturally distinct user groups.

**Deliverables**

- Explainability interaction pattern library
- Culturally adaptive agent interface framework

## II. Inclusio Scalabilis (Scalable, Inclusive Tech Solutions)

**Why this is high-impact**

- Strong path from local interventions to repeatable systems for public impact.

**Focus areas**

- Low-code agent builders for nonprofits
  - Core question: can small organizations configure useful agents without engineers?
  - Method: prototype visual workflow builder for recurring operations.
- Accessibility-first agent design
  - Core question: do agent interfaces meet real assistive-technology needs?
  - Method: WCAG-centered audits and remediation templates.

**Deliverables**

- Minimal nonprofit agent builder blueprint
- Inclusive agent checklist for design and review

## III. Salus Feminarum (AI for Women’s Health)

**Why this is high-impact**

- Direct continuity with your health-oriented research and prototyping track.

**Focus areas**

- Agentic support for underserved groups
  - Core question: can culturally aware agents reduce support inequities?
  - Method: prototype tone and prompt adaptations with anonymous interaction design.
- Privacy-preserving personalization
  - Core question: how to personalize safely under sensitive constraints?
  - Method: experiment with privacy-aware adaptation strategies and constrained retention.

**Deliverables**

- Women’s health agent prototype concept
- Privacy-first personalization protocol

## IV. Cultura et Fiducia (Cross-Cultural Tech Adoption)

**Why this is high-impact**

- Your cross-cultural lens creates differentiated research value.

**Focus areas**

- Trust differences across high-context and low-context norms
  - Core question: which tone, clarity level, and interaction style improve trust by context?
  - Method: A/B tests on phrasing, directness, and explanatory depth.
- Language hybridity in AI interaction
  - Core question: how to preserve accuracy under code-switching?
  - Method: evaluate agent outputs on mixed-language prompts and domain tasks.

**Deliverables**

- Cultural fluency design guide for agent interactions
- Code-switching evaluation set for practical testing

## V. Ethica et Gubernatio (Ethical AI Governance)

**Why this is high-impact**

- Governance quality determines whether agentic systems scale responsibly.

**Focus areas**

- Bias audits for agentic systems
  - Core question: what bias patterns emerge by dataset and deployment context?
  - Method: run structured bias checks using your existing validation matrix logic.
- Participatory AI design
  - Core question: how can communities co-design agents that represent their needs?
  - Method: facilitated workshops with stakeholder feedback loops.

**Deliverables**

- Bias mitigation checklist and test protocol
- Participatory design facilitation guide
