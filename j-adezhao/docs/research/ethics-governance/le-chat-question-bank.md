# Quaestiones Le Chat (Ethics and Governance Bank)

Reference question set for governance-focused Le Chat audits.

Use this bank before validating deployment, safety, and access claims.

## Core Governance Questions

1. What explicit approval gates are required before irreversible actions?
2. What fields are available in audit exports (prompt, action, decision, timestamp)?
3. How are role-based access controls enforced across integrated tools?
4. What retention and redaction policy applies to logs containing sensitive data?
5. How does the system handle policy violations and escalation paths?

## Integration and Policy Questions

1. Which deployment modes are currently supported and under which plan constraints?
2. What interoperability standards are currently stable versus roadmap-stage?
3. Which safeguards prevent prompt injection from connected documents?
4. How should teams benchmark risk-adjusted performance weekly?
