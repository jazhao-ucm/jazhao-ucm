\section*{Probability Models}

\subsection*{Bayesian Update}
\[
P(H\mid E)=\frac{P(E\mid H)P(H)}{P(E)}
\]

Use Bayesian updating to revise agent confidence when new evidence arrives.

\subsection*{State Transition Uncertainty}
\[
P(s_{t+1}\mid s_t,a_t)
\]

This transition kernel represents environment uncertainty under action selection.
