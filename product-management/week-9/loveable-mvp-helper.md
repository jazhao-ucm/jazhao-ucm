# Loveable MVP Helper (Week 9 — Mar 23)

## What to build (the concept)
An app that reduces awkward, repetitive “small talk only” by:
- Starting connection through **structured gamified activities** (shared action)
- Using **common-interest + bucket-list style hooks** to generate conversation naturally
- Using a **quality over quantity** interaction model (fewer, more intentional interactions)
- Making first contact easy via an **“UNFILTERED” Q&A** flow (both people answer the same prompt; answer first to reduce anxiety)
- Nudging toward depth via **deep prompt styles** (e.g., “would you rather” style)

## Core “Factor/Feeling” (success feeling)
Users should feel:
- Safe/comfortable to start talking
- Less stuck in repetitive openers
- That the app leads them toward deeper conversation with the right people

## MVP feature checklist (minimum to match the notes)
### Onboarding + profile inputs
- Interests / what you value
- Deal breakers (what you cannot stand)
- Bucket-list goals (shared goals / activities to do together)
- Personality test results used as matchmaking input for conversation starters
- Profile refresh cycle (refresh weekly / every 2 weeks / monthly as discussed)

### Matching + interest hooks
- **COMMON INTEREST** recommendations (interest matching)
- **BUCKET LIST** recommendations (shared goals / activities)
- Early surfacing of values / deal breakers via signals

### First contact (ice-breaker)
- Start with a gamified activity / structured prompt, not just free chat
- Provide an interaction path where the “next step” is clear

### Conversation mechanics
- **“UNFILTERED” Q&A**:
  - Both people answer the same question/prompt
  - User answers first to reduce anxiety
- **Deep question prompts** beyond shallow greetings
- Quality over quantity:
  - limit interactions / encourage deeper engagement per day

### Journey flow (what screens should support)
- Awareness
- Discovery
- First contact (activity/messages)
- Evaluation (shared values + “vibe”)
- Follow-up (move from first contact to repeated connection)
- Relationship building (repeat interactions OR transition out)

## Screens / prototype flow to implement in Loveable
1. **Onboarding**
2. **Profile setup** (values, interests, deal breakers, bucket list goals, personality test)
3. **Discovery / matching** (interest-based recommendations + “most matching”)
4. **Activity / game start** (structured first contact)
5. **Unfiltered Q&A** (same prompt for both; user answers first)
6. **Conversation depth prompt** (deep prompts to reduce “small talk only”)
7. **Follow-up / next step** (move toward repeated connection)
8. **Weekly reset / refresh** (new prompts + refreshed inputs cycle)
