# Midterm Quick Notes

One-page cheat sheets for each UCM course. **Print only the page for your exam.**

## Files

| File | Course | Print for |
|------|--------|-----------|
| `spanish-only.tex` | Spanish | Spanish exam |
| `strategic-only.tex` | Strategic Management | Strategic exam |
| `product-mgmt-only.tex` | Product Management | Product Mgmt exam |
| `intro-eu-only.tex` | Introduction to EU | Intro EU exam |
| `management-global-only.tex` | Management Global Markets | Mgmt Global exam |
| `blue-notes-all.tex` | All 5 courses | Combined (p.1=Spanish, p.2=Strategic, p.3=Product, p.4=Intro EU, p.5=Mgmt Global) |

## Compile

```bash
pdflatex spanish-only.tex      # Spanish only
pdflatex blue-notes-all.tex   # All 5, then print specific pages
```

## Design

- **If they ask... → Think...** section at top (quick lookup)
- Tables for comparisons (por/para, resources/capabilities, creation/capture)
- 10pt font, green accents, black text
- One page per course
