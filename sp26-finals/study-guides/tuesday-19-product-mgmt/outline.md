## Study Guide Modules

### 1. Product Management Fundamentals
- What is Product Management
- Core PM responsibilities (Vision, Discovery, Definition, Execution, Launch, Measure)
- Essential PM skills (Strategic thinking, analytical, communication, empathy, leadership, technical)
- Common frameworks (RICE, JTBD, Lean Startup, OKRs)

### 2. Design Thinking for Product Managers
- Overview and importance
- 5 Phases:
	1. **EMPATHIZE** - Understand user pain points and motivations
	2. **DEFINE** - Frame the real problem with user perspective
	3. **IDEATE** - Generate many potential solutions
	4. **PROTOTYPE** - Build quick, low-fidelity versions to learn
	5. **TEST** - Get user feedback and iterate
- User research techniques (interviews, surveys, observation, usability testing, analytics)

### 3. Value Creation and Value Capture
- Value proposition development
- Five ways to create value (differentiation, efficiency, quality, experience, community)
- Pricing strategies (cost-plus, value-based, penetration, skimming, freemium, subscription)
- Business model monetization

### 4. Business Model Canvas
- 9 Canvas blocks explained:
	1. Customer Segments
	2. Value Propositions
	3. Distribution Channels
	4. Customer Relationships
	5. Revenue Streams
	6. Key Resources
	7. Key Activities
	8. Key Partnerships
	9. Cost Structure
- How to fill out the canvas
- Testing assumptions

### 5. Prototyping and Testing
- Prototype fidelity levels (sketch → wireframe → low-fi digital → high-fi mockup → functional)
- When to use each fidelity level
- Rapid prototyping tips (low-fi first, build to learn, focus on one variable)
- User testing approaches (moderated, unmoderated, A/B testing)
- How to ask good test questions

### 6. Launch Planning and Execution
- Three launch phases:
	- **PRE-LAUNCH** (6-12 weeks): Build anticipation, coordinate teams, prepare materials
	- **LAUNCH DAY**: Press release, social blast, email campaigns, monitor
	- **POST-LAUNCH** (30 days): Gather feedback, iterate, track metrics, build momentum
- Go-to-Market (GTM) planning
- GTM template elements (Who, What, Where, When, How, Success Metrics)

### 7. Product Metrics and Analytics
- North Star metric (one metric that represents success)
- Key product metrics families:
	- **ACQUISITION** - Signups, CAC, conversion rate
	- **ACTIVATION** - Onboarding completion, time to first action
	- **RETENTION** - DAU/WAU/MAU, churn rate, repeat rate
	- **REVENUE** - MRR, ARPU, LTV
	- **REFERRAL** - Sharing rate, NPS, viral coefficient
- Data-driven decision making
- Dashboard and reporting

### 8. Case Study: Zara Fast Fashion
- The challenge: Traditional fashion retail moves too slowly
- Innovation: Fast fashion loop (observe → design → produce → iterate → repeat)
- Speed and data-driven culture as competitive advantage
- Key success factors: vertical integration, data collection, small batches, rapid feedback loops
- PM lessons: Speed beats perfection, real data beats market research, small tests de-risk

### 9. Key Terms Glossary
- Product-Market Fit
- MVP (Minimum Viable Product)
- User Stories
- Wireframe vs Prototype
- User Research
- A/B Testing
- CAC (Customer Acquisition Cost)
- Churn Rate & Retention
- Market Segmentation
- Pivot
- Go-to-Market
