\footnotesize

\textbf{\color{eublue}A. Internationalization basics}
\begin{itemize}
\item International business = operations across borders.
\item Why go abroad: market, resources, efficiency, learning.
\item Failure drivers (course): wrong location; weak CA; poor execution.
\item ``Location is the masterpiece''.
\end{itemize}

\textbf{\color{eublue}B. Complexity + specialization}
\begin{itemize}
\item Economic complexity = raw $\to$ complex products (skills + know-how).
\item Key idea (class): be good at something somewhere (specialization).
\item Complexity without buyer understanding = problem (sellability).
\end{itemize}

\textbf{\color{eublue}C. Resources, capabilities, CA}
\begin{itemize}
\item Resources = what you have; Capabilities = what you do.
\item Competitive advantage = better than rivals; Absolute CA = rare.
\item Before abroad: CA strong enough? If not, high risk.
\end{itemize}

\textbf{\color{eublue}D. Four phases (location risk)}
\begin{itemize}
\item Conceive $\to$ Execute $\to$ Establish $\to$ Expand.
\item Execute = highest investment; wrong place = fail.
\item Establish = most dangerous (many fail).
\end{itemize}

\textbf{\color{eublue}E. Culture + risks}
\begin{itemize}
\item Culture shock: honeymoon $\to$ hard $\to$ adapt.
\item Artificial risk (laws) vs natural risk (language/culture).
\item ``You cannot escape language.''
\end{itemize}

\textbf{\color{eublue}F. HRM (skills, training, motivation)}
\begin{itemize}
\item Skills can be built: training + hiring.
\item Training formats: classroom, discussion, conferences.
\item Job rotation: finance $\to$ production $\to$ sales (big corporations).
\item Motivation first; without it training = useless.
\item Habit: ask questions; asking $\neq$ weakness; avoids costly mistakes.
\end{itemize}

\textbf{\color{eublue}G. Export vs Int'l vs FDI}
\begin{itemize}
\item Export = product moves; Int'l = company operations move.
\item FDI = long-term commitment $\to$ HRM complexity.
\end{itemize}

\textbf{\color{eublue}H. Macro blocks + current context}
\begin{itemize}
\item Blocks discussed in class: EU (27), USA, Asia block (ASEAN idea), S. America block (idea).
\item Europe dependence: imports/energy $\to$ price shocks.
\item UN Security Council P5 (mentioned): USA, China, Russia, France, UK.
\end{itemize}

\textbf{\color{eublue}I. Rapid definitions (1 line each)}
\begin{itemize}
\item Globalization = freer movement, interconnected economy.
\item Slowbalization = slowing, protectionism, localization.
\item Supply chain = parts move across borders before final product.
\item Localization = adapt product to local market before scaling abroad.
\end{itemize}
