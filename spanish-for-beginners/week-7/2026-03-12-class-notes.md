# Spanish for Beginners - Class Notes
## March 12, 2026 - Week 7

**Focus:** Unit 4 grammar: `tener`, `tener que`, `preferir`, `ir` + planes; `de` for objects

---

I. Grammar `Tener` / `Tener que` / `Preferir`
   A. `Tener` (presente)
      1. (yo) tengo
      2. (tú) tienes
      3. (él/ella/usted) tiene
      4. (nosotros/nosotras) tenemos
      5. (vosotros/vosotras) tenéis
      6. (ellos/ellas/ustedes) tienen
   B. `Tener que` (obligación)
      1. Yo tengo que + infinitivo
      2. Tú tienes que + infinitivo
      3. Él/Ella/Usted tiene que + infinitivo
      4. Nosotros/Nosotras tenemos que + infinitivo
      5. Vosotros/Vosotras tenéis que + infinitivo
      6. Ellos/Ellas/Ustedes tienen que + infinitivo
   C. `Preferir` (presente)
      1. Yo prefiero
      2. Tú prefieres
      3. Él/Ella/Usted prefiere
      4. Nosotros/Nosotras preferimos
      5. Vosotros/Vosotras preferís
      6. Ellos/Ellas/Ustedes prefieren

II. Grammar `Ir` (presente) + planes
   A. `Ir` (presente)
      1. Yo voy
      2. Tú vas
      3. Él/Ella/Usted va
      4. Nosotros/Nosotras vamos
      5. Vosotros/Vosotras vais
      6. Ellos/Ellas/Ustedes van
   B. Estructuras de planes
      1. `Voy a` + lugar
         a. A la playa
         b. A clase de español
      2. `Voy al` + lugar
         a. Al gimnasio
      3. `Voy de` + actividad
         a. De excursión
         b. De compras
         c. De viaje

III. `De` objetos (formas tipo: X de Y)
   A. Combinaciones del ejercicio
      1. Toalla de playa
      2. Carne de identidad
      3. Pasta de dientes
      4. Gafas de sol
      5. Tarjeta de crédito
      6. Gel de ducha
      7. Cargador de móvil

IV. Diálogo frases para describir ropa
   A. Calidad / precio
      1. Es bonito
      2. Es barato
      3. Es súper barato
      4. También es bastante barato/a
      5. Es muy sencillo/a
      6. No está mal
      7. Está muy bien
   B. Talla / preferencia
      1. Es talla L
      2. Yo tengo la M
      3. Yo prefiero la rosa
      4. Creo que la compro

X. 30-Second Cold Call Version.
A. 3-Line Speaking Frame.
   1. TOPIC: STATE THE CORE QUESTION IN ONE LINE.
   2. LOGIC: GIVE 2 DRIVERS/FACTORS THAT EXPLAIN THE ANSWER.
   3. ACTION: END WITH ONE DECISION OR IMPLICATION.
B. Delivery Rule.
   1. 25-30 SECONDS MAX.
   2. NO BACKGROUND STORY; GO STRAIGHT TO CLAIM + EVIDENCE + CLOSE.

CI. Oral Answer Template (60-90 Seconds, Class Use).
A. Opening.
   1. STATE THE CORE QUESTION OR TOPIC IN ONE LINE.
   2. STATE YOUR MAIN CLAIM/THESIS IN ONE LINE.
B. Logic.
   1. GIVE 2-3 KEY FACTORS OR DRIVERS.
   2. LINK EACH FACTOR TO THE CLAIM (CAUSE -> EFFECT).
C. Action Or Implication.
   1. END WITH ONE DECISION, RECOMMENDATION, OR CONSEQUENCE.
   2. CLOSE CLEARLY WITHOUT ADDING NEW IDEAS.
D. Delivery Rule.
   1. 60-90 SECONDS.
   2. CLAIM -> EVIDENCE -> CLOSE.
