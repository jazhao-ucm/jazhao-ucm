I. Ropa `La prenda es...` (cualidad)
   A. Ejemplos del ejercicio
      1. Es sencilla
      2. Es cómoda
      3. Es estampada
      4. Es elegante
      5. Es fácil de llevar

II. Ropa `La prenda es de...` (tipo/diseño/material)
   A. Partes / diseños
      1. De rayas
      2. De tirantes
      3. De manga corta
      4. De manga larga
      5. De algodón
      6. Liso/a
      7. Hombre / mujer
   B. Nota ejercicio
      1. Concordancia liso/a, blanco/a, etc. (fem/masc)

IV. Combinar palabras (ejercicio de máximo número)
   A. Plantilla (lo que se repite)
      1. Prenda: `un`/`una`/`unos`
      2. Rasgo: adjetivos (original, elegantes, bonitos, rojos, negros)
      3. `de` + rasgo (de tirantes, de algodón, de rayas)
      4. Largo/cortos
   B. Ejemplos de combinaciones (según tu práctica)
      1. Un vestido: original; de tirantes; de algodón; de rayas; largo
      2. Una camiseta: estampada; de manga larga; de algodón
      3. Unos pantalones: elegantes; bonitos; rojos; de rayas; de algodón; cortos
      4. Unos zapatos: elegantes; bonitos; negros

X. 30-Second Cold Call Version.
A. 3-Line Speaking Frame.
   1. TOPIC: STATE THE CORE QUESTION IN ONE LINE.
   2. LOGIC: GIVE 2 DRIVERS/FACTORS THAT EXPLAIN THE ANSWER.
   3. ACTION: END WITH ONE DECISION OR IMPLICATION.
B. Delivery Rule.
   1. 25-30 SECONDS MAX.
   2. NO BACKGROUND STORY; GO STRAIGHT TO CLAIM + EVIDENCE + CLOSE.

XI. Oral Answer Template (60-90 Seconds, Class Use).
A. Opening.
   1. STATE THE CORE QUESTION OR TOPIC IN ONE LINE.
   2. STATE YOUR MAIN CLAIM/THESIS IN ONE LINE.
B. Logic.
   1. GIVE 2-3 KEY FACTORS OR DRIVERS.
   2. LINK EACH FACTOR TO THE CLAIM (CAUSE -> EFFECT).
C. Action Or Implication.
   1. END WITH ONE DECISION, RECOMMENDATION, OR CONSEQUENCE.
   2. CLOSE CLEARLY WITHOUT ADDING NEW IDEAS.
D. Delivery Rule.
   1. 60-90 SECONDS.
   2. CLAIM -> EVIDENCE -> CLOSE.
